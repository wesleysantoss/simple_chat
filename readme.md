## **Simple Chat**
Chat simples, feito com o objetivo de treinar e estudar mais sobre NodeJS.
Para essa aplicação foi utilizado o Express, Socket.io e MongoDB.

## **Desenvolvimento**

 

    $ npm install
    $ mongod
    $ npm start
